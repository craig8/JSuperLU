JSuperLU
========

A Java port of 
[SuperLU MT](http://crd-legacy.lbl.gov/~xiaoye/SuperLU/#superlu_mt) - a 
multi-threaded, direct solver of sparse, complex, nonsymmetric systems 
of linear equations.
